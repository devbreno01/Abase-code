Sistema de Biblioteca (Herança)

Crie uma classe base chamada ItemBiblioteca.

Atributos
- title
- year

Método
getInfo()

Depois crie duas classes que herdam dessa classe:

-Livro
-Revista

Requisitos

Livro deve ter:

author

Revista deve ter:

edition

Cada classe deve sobrescrever o método getInfo().
